# 简易通讯录 (ContactBook)
一个基于 C++ 控制台的简易通讯录项目，适合 C++ 初学者进阶练习。

## 项目功能
- 添加联系人（姓名 + 电话）
- 显示所有联系人
- 按姓名查找联系人
- 按编号删除联系人
- 文件持久化存储（关闭程序不丢失数据）
- 支持中文姓名与中文路径

---

## 本项目主要学习内容
通过该项目，你将系统掌握以下 C++ 核心知识点：

### 1. 面向对象编程（OOP）
- 类与对象的设计与使用
- 封装思想：数据私有化 + 公开接口
- 成员函数、私有变量的实际应用

### 2. 数据结构与容器
- 使用 `vector` 动态存储联系人列表
- 理解动态数组的增删查改
- 掌握常见容器遍历方法

### 3. 文件操作（持久化存储）
- 文件读取 `ifstream`
- 文件写入 `ofstream`
- 程序启动自动加载数据
- 退出前自动保存数据
- 学会制作“数据不丢失”的软件

### 4. 字符串与输入输出
- string 字符串处理
- 控制台菜单交互
- 标准输入输出 cin / cout
- 中文控制台编码适配

### 5. 程序逻辑与模块化开发
- 清晰的程序结构（头文件 + 源文件）
- 函数拆分与业务逻辑解耦
- 菜单循环与用户交互
- 工程化文件结构

### 6. 工程化构建
- CMake Lists 编写
- 多文件项目编译
- Release 发布版本打包
- 可移植软件制作（全新电脑可运行）
